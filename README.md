# Rock Paper Scissors

A simple browser-based Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## Preview


## Features

- Choose Rock, Paper, or Scissors
- Random computer move
- Displays game result instantly
- Tracks wins, losses, and ties using `localStorage`
- Reset score button

## Files

- `rock-paper-scissors.html` - game UI and page structure
- `rock-paper-scissors.css` - visual styling
- `rock-paper-scissors.js` - game logic and score tracking
- `Public/` - image assets used in the game

## How to run

1. Open `rock-paper-scissors.html` in a web browser.
2. Click one of the move buttons to play.
3. Use the Reset Score button to clear the score.

## Notes

- Score is saved in browser `localStorage`, so it persists across page refreshes.
- The game is meant for local use and works without any additional build steps.
